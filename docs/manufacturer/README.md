# Документация производителя

Эта папка содержит оригинальные файлы от производителя датчика давления.

## ⚠️ Важное предупреждение

**Документация производителя содержит очень мало полезной информации.**

Все практические знания о работе с датчиком были получены **экспериментальным путём** в ходе длительных тестов. Документация производителя не содержит:

- Правильного описания протокола работы
- Информации о команде `0xAC`
- Корректных данных о калибровке
- Реальных ограничений частоты обновления
- Описания статус-байтов

## Рекомендации

**Используйте примеры кода и описания из основного репозитория** как основной источник информации:

- [Прямое подключение к ESP32](../../examples/direct_connection/README.md)
- [Работа через PCA9548A](../../examples/pca9548a_multiplexer/README.md)

Файлы в этой папке размещены **только для справки** и могут быть полезны для понимания базовых характеристик датчика, но не для практической работы с ним.

## Содержимое папки

- `KY-3V3-IIC-10Bar-B.pdf` — PDF документация (китайский/английский)
- `KY-3V3-IIC-16Bar-B.pdf` — PDF документация для 16 bar версии
- `1200.C程序CAL_H 与CAL_L的计算.docx` — расчёт калибровочных значений
- `I2C数字输出校准值应用.docx` — применение калибровочных значений
- `reading pressure.docx` — чтение давления
- `压力换算0_1Mpa_1.6Mpa举例008.docx` — примеры пересчёта давления
- `KSW-IIC传感器示列程序.rar` — примеры программ (архив)
- `STM32-testI2C.rar` — тестовые программы для STM32 (архив)
